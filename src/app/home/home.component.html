About this site

This site was built by a Helio Training team of four. The project saw many versions until we got to final product of this store. Angular is an extremely powerful programming language which can be a double edged sword. Our biggest hiccup we ran into was getting the HTTP Client to run properly. The finished product however makes good use of the following:

  • *ngModal
  • HTTP Client
  • *ngFor
  • Reactive Forms
  • Routing
  • Services
  • Modals
  • API with Hapi & Mongoose
  • Database with MongoDB
  • Styling with MDBootstrap
